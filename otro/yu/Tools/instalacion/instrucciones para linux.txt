instrucciones para ejecutarlo en linux:

abres la terminal
primero debemos darle permisos al archivo como super usuario usando el siguiente comando:

sudo su

luego de haber puesto el comando te pedirá tu contraseña
para luego lanzar el siguiente comando:

cd /opt/lampp/htdocs/yu/Tools/cache/
chmod +x borrar_caché_linux.sh

y ya en la misma terminal, deberás lanzar este proximo comando:

./borrar_caché_linux.sh

ya lo que queda es esperar a que se ejecute el script
